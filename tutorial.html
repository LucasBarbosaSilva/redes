Tutorial

  1. Procurar um site para a hospedagem e domínio:

    Na internet temos várias opções de servidores para hospedagem de sites, uns pagos, outros gratuitos. 000webhost e infinityfree são exemplos de sites gratuitos para hospedagem. Neste tutorial vamos usar o infinityfree.net, portanto faça login neste site. O servidor infinityfree também nos pede um domínio para o nosso site, para isso vamos usar o freenom para procurar o domínio.
    Assim faça login nos dois sites antes de prosseguir com o tutorial.

  2. Conseguindo um domínio:

    Para conseguir um domínio gratuito no freenom, insira um nome de sua preferência e clique em "Verificar ", verifique quais as opções de domínio, adcione uma ao carrinho e conclua a aquisição.

  3. Configurando o domínio para o servidor do InfinityFree:

    Vamos agora finalizar o cadastro do domínio. Acesse:My Domains > Manage Domain > Management Tools > Nameservers > Utilize a opção "custom nameservers" e preencha os dois primeiros campos com "ns1.epizy.com" e "ns2.epizy.com" Clique em Change Nameservers para salvar;

  4. Criando a conta no infinityfree:

    No site do infinityfree clique em New Account > utilize a opção "custom domain" e preencha o campo com o domínio escolhido no Freenom; Após isso, preencha (se quiser) a opção "Account Password" e clique em "Create Account"; Para finalizar e ativar a conta clique em "Open Control Painel" > "I Approve". Pronto servidor configurado para receber o upload.

  5. Fazendo o upload com o cliente FTP

    Para fazer o upload para o servidor vamos utilizar um cliente FTP (protocolo da camada de transporte). Existem vários clientes, vamos utilizar um online chamado onlineftp.ch. Preencha os campos de acordo com as informações de usuário fornecidas pelo InfinityFree, faça o login e o upload dos arquivos do seu site. Assim o cliente colocou no servidor o seu site e agora ele está disponível para o acesso na internet.
